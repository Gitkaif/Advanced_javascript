# Advanced_js

A collection of advanced JavaScript examples, utilities and small projects focused on modern patterns, performance, and tooling.

## Contents
- Examples demonstrating advanced language features (async/await, generators, proxies)
- Utilities for performance and memory profiling
- Tooling configs (ESLint, Prettier, build scripts)
- Tests and CI-friendly setup

## Prerequisites
- Node.js 16+ (or current LTS)
- npm or yarn

## Installation
```bash
git clone <repo-url>
cd Advanced_js
npm install
```

## Available scripts
- `npm run build` — build project (if applicable)
- `npm run start` — run demo or example server
- `npm test` — run tests
- `npm run lint` — run linter and fixable rules

## Project structure
- /src — source code and examples
- /examples — runnable example apps
- /tests — unit/integration tests
- /scripts — build and utility scripts
- README.md — this file
